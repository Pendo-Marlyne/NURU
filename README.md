# Nuru MedFinTech

**Illuminating Connected Care Across East Africa**

Nuru is a patient-centered medfintech platform that removes financial barriers to life-saving healthcare in East Africa. By connecting patients, hospitals, verified donors, insurers, and financial partners, Nuru enables faster funding, transparent tracking, and better outcomes for complex treatments — especially transplants, oncology, fertility, high-risk pregnancies, and vision care.

Built with empathy, security, and transparency at its core.

## ✨ Key Features

- **For Patients & Families**
  - Quick eligibility check for funding support
  - Real-time dashboard: see funds raised, donor matches, and next steps
  - Clear, jargon-free guides and cost breakdowns
  - Community support groups and care navigators
  - Secure, encrypted personal data (Kenya DPA & HIPAA-aligned)

- **For Hospitals & Providers**
  - Same-day fund disbursement (M-Pesa/bank)
  - Revenue dashboard with real-time claims & donor tracking
  - API-first EHR integration
  - 85% faster claims processing
  - Donor matching for unfunded high-cost cases

- **For Donors, NGOs & Investors**
  - Verified, transparent impact tracking
  - Blockchain-secured transaction flows
  - Real-time donor-to-patient visibility
  - Strategic partnership opportunities

- **Core Platform Capabilities**
  - End-to-end encryption (AES-256)
  - Multi-factor authentication & role-based access
  - Fraud detection & transaction monitoring
  - Instant donor matching & micro-financing options

## 🛠️ Tech Stack

- **Frontend**: HTML5, Tailwind CSS (via CDN), vanilla JavaScript
- **Icons**: Font Awesome 6
- **Animations**: Custom fade-in via Intersection Observer
- **Design System**: Glassmorphism + warm navy/gold theme
- **Responsive**: Mobile-first (hamburger menu with toggle)
- **No build tools** — pure static HTML/CSS/JS (easy to host on GitHub Pages, Netlify, Vercel)
